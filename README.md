# Rohan Chakraborty — Portfolio CV Website

A dark, cinematic portfolio website built with HTML, CSS, and vanilla JavaScript.

## ✨ Features

- **Animated Loader** — Smooth percentage counter preloader
- **Three.js Particle Background** — Live floating particle field
- **Custom Magnetic Cursor** — Reactive cursor with lag ring
- **Lenis Smooth Scrolling** — Buttery smooth scroll experience
- **GSAP Scroll Animations** — Reveal animations on scroll
- **Hero Name Glitch Effect** — Hover the name for a glitch animation
- **Live IST Clock** — Real-time Kolkata time in sidebar
- **Animated Skill Bars** — Progress bars that animate into view
- **Infinite Marquee** — Contact section scrolling text
- **One-click Email Copy** — Copy email to clipboard
- **Fully Responsive** — Mobile-first design with hamburger menu

## 🛠 Tech Stack

- HTML5 / CSS3 (Vanilla — no frameworks)
- JavaScript (Vanilla)
- [GSAP 3](https://greensock.com/gsap/) — Animations
- [Three.js](https://threejs.org/) — WebGL particle background
- [Lenis](https://github.com/studio-freight/lenis) — Smooth scrolling
- Google Fonts (Syne + Space Mono)

## 🚀 Deployment (GitHub Pages)

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://yourusername.github.io/repo-name/`
